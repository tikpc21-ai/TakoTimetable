# คู่มือการใช้งานระบบตารางเรียน-ตารางสอน TakoTimetable (อัปเดต)
### โรงเรียนวัดบ้านตะโกตาพิ

ระบบจัดการตารางเรียนตารางสอน TakoTimetable ได้รับการพัฒนาเป็นเว็บแอปพลิเคชันแบบ Single Page Application (SPA) ที่ทำงานบนหน้าเว็บเบราว์เซอร์ได้ทันทีโดยไม่จำเป็นต้องเชื่อมต่ออินเทอร์เน็ต ระบบนี้รองรับสองภาคเรียน (เทอม 1 และ 2) แยกขาดจากกัน และมาพร้อมระบบล็อกรหัสผ่านความปลอดภัยวิชาการ

---

## 🔒 1. สิทธิ์การใช้งานตาราง (Public vs Admin Access)

เพื่อป้องกันตารางเรียนเสียหายจากการแก้ไขโดยไม่ตั้งใจ ระบบจึงแบ่งมุมมองออกเป็น 2 ส่วน:
*   **สำหรับนักเรียนและครูทั่วไป (Public View)**:
    *   เปิดดูตารางเรียนรายห้อง ตารางสอนครูรายบุคคล และตารางรวมประจำวันได้ทันที ค้นหาผ่านกล่องค้นหาด่วนได้สะดวก
    *   *เมนูจัดตาราง เมนูตั้งค่า เมนูวิชาเรียน และปุ่มนำเข้าข้อมูลจะถูกซ่อนไว้โดยอัตโนมัติ*
*   **สำหรับฝ่ายบริหารงานวิชาการ (Admin View)**:
    *   กดปุ่ม **"เข้าสู่ระบบวิชาการ"** (รูปแม่กุญแจ) บริเวณด้านล่างของแถบเมนูซ้าย
    *   ป้อนรหัสผ่านเข้าถึงสิทธิ์: **`1234`** (คุณครูสามารถเข้าไปสลับรหัสผ่านใหม่นี้ได้ในหน้า "ตั้งค่าระบบ")
    *   เมื่อเข้าสู่ระบบแล้ว เมนูการจัดสอนแทน, วางแผนตารางเรียน (Planner), จัดการรายวิชา และตั้งค่าระบบ จะปรากฏให้ใช้งานทันที
    *   กด **"ออกจากระบบวิชาการ"** ทุกครั้งเมื่อตั้งค่าเสร็จสิ้น เพื่อกลับเข้าสู่โหมดใช้งานปกติสำหรับสาธารณะ

## 🚀 2. การนำเข้าข้อมูลตารางเรียนและหลักสูตร

เมื่อเข้ามาในโหมดวิชาการ ปุ่ม **"นำเข้าข้อมูล Excel"** จะปรากฏขึ้นขวาบน เมื่อกดปุ่มจะมีตัวเลือกการนำเข้า 2 รูปแบบ:

### ก) การนำเข้าตารางเรียนหลัก (ตารางเรียนประจำวัน)
*   **การเลือกเทอม**: เลือกภาคเรียนเป้าหมายที่คุณต้องการเขียนข้อมูลทับ (ภาคเรียนที่ 1 หรือ ภาคเรียนที่ 2)
*   **การเลือกไฟล์**: ลากและวางไฟล์ตารางสอนหลัก เช่น `โปรแกรมช่วยจัดการตารางสอน v.4.6 1-2569 v6.xlsm` เข้ามาในช่องอัปโหลด
*   ระบบจะทำการดึงข้อมูลครูวิชาการ, ตารางรวม 8 คาบ 5 วันของประถมและมัธยมแยกเทอมกันอย่างเด็ดขาด

### ข) การนำเข้าโครงสร้างรายวิชาเรียน (จากหลักสูตร)
*   **ทำไมต้องนำเข้า**: เพื่อหลีกเลี่ยงการมานั่งคีย์วิชาเรียนใหม่ทีละวิชา เจ้าหน้าที่วิชาการสามารถสกัดวิชาเรียนทั้งหมดของโรงเรียนเข้ามาในระบบได้ทันที
*   **การเลือกไฟล์**: คลิกแท็บ **"นำเข้าโครงสร้างรายวิชา"** ในหน้าต่างอิมพอร์ต แล้วลากไฟล์โครงสร้างรายวิชาของโรงเรียน เช่น `TKP_โครงสร้างเรียนมัธยม 2569.xlsx` มาวาง
*   ระบบจะประมวลผลชีทเวลาเรียนดึงรหัสวิชาและชื่อรายวิชาของ ม.1 - ม.3 ทุกตัว ทั้งวิชาพื้นฐานและเพิ่มเติม เข้ามาเป็นลิสต์วิชาเรียนพร้อมใช้ในการจัดตาราง

## 📶 3. การใช้งานออนไลน์ (Online) และออฟไลน์ (Offline)

### ก) การใช้งานแบบออฟไลน์ (Offline-First)
*   ระบบตารางเรียนทำงานได้สมบูรณ์โดย**ไม่ต้องเชื่อมต่ออินเทอร์เน็ต** ข้อมูลจะถูกจัดเก็บไว้ใน LocalStorage บนเครื่องคอมพิวเตอร์ของคุณอัตโนมัติ
*   หากต้องการเปิดแชร์ตารางสอนให้คุณครูท่านอื่นหรือนักเรียนสามารถสแกนเข้ามาดูผ่านหน้าจอมือถือของตนเองได้จากภายในโรงเรียน:
    1. ดับเบิ้ลคลิกไฟล์ [run_server.ps1](file:///d:/โรงเรียนวัดบ้านตะโกตาพิ/งานวัดผล วิชาการ/ปีการศึกษา 2569/ตารางสอน/run_server.ps1)
    2. จะปรากฏลิงก์ระบุที่อยู่ของเครื่องคอมพิวเตอร์ของคุณในวงแลน เช่น:
       `Local Network (Wi-Fi): http://192.168.1.15:8080/`
    3. นำลิงก์นี้ไปแปลงเป็น QR Code หรือส่งต่อเข้าไลน์กลุ่ม เพื่อให้ทุกคนที่เชื่อมต่อ Wi-Fi โรงเรียนสามารถเปิดดูตารางเรียนได้ทันที

### ข) การใช้งานแบบออนไลน์ (Online Deployment)
*   เนื่องจากระบบถูกเขียนขึ้นเป็นแบบ Client-Side Single Page Application (SPA) คุณครูสามารถนำเอาไฟล์ในโฟลเดอร์นี้ ได้แก่ `index.html`, `app.css`, และ `app.js` อัปโหลดขึ้นโฮสติ้งเปิดสาธารณะได้ฟรีทันที เช่น **GitHub Pages**, **Netlify**, หรือ **Vercel**
*   เมื่อนำขึ้นระบบออนไลน์แล้ว คุณครูและนักเรียนจะสามารถสแกนตรวจสอบตารางเรียนตารางสอน และวิเคราะห์การสอนแทนได้จากมือถือและแท็บเล็ตของตนเองจากที่บ้านหรือที่ใดก็ได้ทั่วโลก 24 ชั่วโมงต่อวัน
